# Themes
## Terminologies
<!-- TODO [EN]==>[FR] -->
+ **Surface** – Backgrounds || large, low-emphasis areas of the screen.
+ **Primary, Secondary, Tertiary** – Accent color roles used to emphasize or de-emphasize foreground elements.
+ **Container** – Fill color for foreground elements like buttons. SHOULD NOT be used for text or icons.
+ **On** – Color for text or icons *on top* of its paired parent color.
	> For example, **on primary** is used for text and icons against the **primary** fill color.
+ **Variant** – Lower emphasis alternative to its non-variant pair.
	> For example, **outline variant** is a less emphasized version of the **outline** color.

## Descriptions
+ **surface-container**: Couleur de fond des barres de navigation
+ **surface**: Couleur de fond par défaut
+ **on-surface**: Couleur par défaut des textes
+ **on-surface-variant**: Couleur atténuée des textes
+ **surface-inverse**: Couleur de fond pour notification snackbar
+ **on-surface-inverse**: Couleur par défaut du texte sur notification snackbar
+ **primary**: Couleur des titres, contrôles & éléments actifs
+ **on-primary**: Couleur du texte sur les éléments actifs (Exple: les filtres)
+ **primary-container**: Couleur de fond pour boutons primaires (surélevés)
+ **on-primary-container**: Couleur du texte sur les boutons primaires
+ **primary-inverse**: Couleur par défaut des textes CTA sur snackbar

+ **secondary**: Fond par défaut des éléments surlignés/interactifs inactifs
+ **on-secondary**: Couleur du texte sur les éléments interactifs inactifs
+ **secondary-container**: Fond par défaut des boutons secondaires
+ **on-secondary-container**: Couleur du texte sur les boutons secondaires
+ **outline**: Couleur de bordures des composants simples(bouton, input, etc.)
+ **outline-variant**: Couleur des lignes de séparation et autres bordures
